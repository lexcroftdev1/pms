Project Summary

This would essentially allow a user to change the text outputs (in the date formats that output text like "Sunday, January 13, 2013".
I would like a link at the bottom that says: Change days and months into different language Then it will basically show two columns.
in column one will be what we currently call the field (this will be regular text on the page).
In column two will be text fields already prepopulated with what we call the field... but editable so they can change it to whatever they want.

Something like

Months

February >> [February field]
March >> [March field]

All 12 months

Days:

Monday >> [Monday field]

All 7 days

Obviously, it should support special characters and other language characters so please make sure all characters are escaped properly as to not break the script/functionality.
